# Art Contest Website

A website for running an art contest. Artists submit their artwork through a form, an admin reviews and approves submissions, and approved artwork is displayed in a public gallery.

## Features

- **Public submission form** — Artists upload their artwork with their name, title, and optional contact info
- **Admin dashboard** — Password-protected page to review submissions, approve/reject them, toggle featured status, and upload artwork on behalf of others (for email/WhatsApp submissions)
- **Gallery** — Featured artwork displayed prominently at the top, all approved artwork in a grid below
- **Email notifications** — Admin receives an email whenever a new submission comes in

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/) (Database + Image Storage)
- [Resend](https://resend.com/) (Email notifications)
